# Deskwork

Free resume, cover letter, invoice and letter builder.

## Deploy to Vercel (recommended — free)

1. Create a free GitHub account (github.com) if you don't have one.
2. Create a new repository, e.g. "deskwork".
3. Upload ALL files in this folder to that repo (keep the folder structure: `src/` folder, `index.html`, `package.json`, etc.)
4. Go to vercel.com, sign in with GitHub.
5. Click "Add New Project" → select your `deskwork` repo.
6. Vercel auto-detects Vite — just click "Deploy".
7. Wait ~1 minute. You'll get a live URL like `deskwork-yourname.vercel.app`.

## Adding a custom domain

In Vercel: Project → Settings → Domains → add your purchased domain (e.g. from Namecheap, GoDaddy) and follow the DNS instructions shown.

## Getting found on Google

1. Go to Google Search Console (search.google.com/search-console), add your site, verify ownership (Vercel shows you how).
2. Submit your homepage URL for indexing.
3. Add more text content to the homepage describing what the tool does — Google needs real text to rank you, especially for searches like "free resume builder Zambia".
